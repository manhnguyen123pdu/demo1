CÂU LỆNH IF

1.2.1. Dạng 1: Câu lệnh if

Định nghĩa: Câu lệnh if dùng để kiểm tra một điều kiện, nếu điều kiện đúng (true) thì đoạn mã trong khối if sẽ được thực thi.

Cú pháp
if (<điều_kiện>) {
    <khối_lệnh>;
}

Lưu đồ thuật toán câu lệnh IF

IfExample.java
public class IfExample {
    public static void main(String[] args) {
        int number = 5;
        
        if (number > 0) {
            System.out.println("Số dương");
        }
        
        System.out.println("Chương trình kết thúc");
    }
}

Giải thích chi tiết:

  • BƯỚC 1: Khai báo biến number = 5
  • BƯỚC 2: Kiểm tra điều kiện number > 0 → 5 > 0 → TRUE
  • BƯỚC 3: Thực thi khối lệnh trong if → in "Số dương"
  • BƯỚC 4: Tiếp tục chương trình → in "Chương trình kết thúc"

CÂU LỆNH IF...ELSE

1.2.2. Dạng 2: Câu lệnh if...else

Định nghĩa: if...else là câu lệnh điều kiện có 2 nhánh: Nếu điều kiện đúng → chạy khối lệnh trong if, nếu sai → chạy khối lệnh trong else.

Cú pháp
if (<điều_kiện>) {
    <khối_lệnh1>;
} else {
    <khối_lệnh2>;
}

Lưu đồ thuật toán câu lệnh IF

IfElseExample.java
public class IfElseExample {
    public static void main(String[] args) {
        double score = 4.5;
        if (score >= 5) {
            System.out.println("Đậu");
        } else {
            System.out.println("Rớt");
        }
        System.out.println("Kết thúc kiểm tra");
    }
}

Giải thích chi tiết:

  • BƯỚC 1: Khai báo biến score = 4.5
  • BƯỚC 2: Kiểm tra điều kiện score >= 5 → 4.5 >= 5 → FALSE
  • BƯỚC 3: Nhảy đến khối else (vì điều kiện sai)
  • BƯỚC 4: Thực thi khối else → in "Rớt"
  • BƯỚC 5: Tiếp tục chương trình → in "Kết thúc kiểm tra"

Kết quả:
Rớt
Kết thúc kiểm tra

CÂU LỆNH IF...ELSE IF...ELSE

1.2.3. Dạng 3: Câu lệnh if...else if...else

Định nghĩa: Cho phép kiểm tra nhiều điều kiện khác nhau, theo thứ tự từ trên xuống. Khi gặp điều kiện đúng đầu tiên, chương trình sẽ thực hiện và không kiểm tra tiếp.

Cú pháp
if (<điều_kiện1>) {
    <khối_lệnh1>;
} else if (<điều_kiện2>) {
    <khối_lệnh2>;
} else if (<điều_kiện3>) {
    <khối_lệnh3>;
} else {
    <khối_lệnh4>;
}

Lưu đồ thuật toán câu lệnh IF

IfElseIfExample.java
public class IfElseIfExample {
    public static void main(String[] args) {
        double score = 8.5;
        String grade;
        
        if (score >= 9) {
            grade = "Xuất sắc";
            System.out.println("Chúc mừng! Bạn đạt loại: " + grade);
        } else if (score >= 7) {
            grade = "Khá";
            System.out.println("Tốt! Bạn đạt loại: " + grade);
        } else if (score >= 5) {
            grade = "Trung bình";
            System.out.println("Bạn đạt loại: " + grade);
        } else {
            grade = "Yếu";
            System.out.println("Cần cố gắng! Bạn đạt loại: " + grade);
        }
        
        System.out.println("Kết thúc xếp loại");
    }
}

Giải thích chi tiết:

  • BƯỚC 1: Khai báo biến score = 8.5
  • BƯỚC 2: Kiểm tra điều kiện 1: score >= 9 → 8.5 >= 9 → FALSE
  • BƯỚC 3: Kiểm tra điều kiện 2: score >= 7 → 8.5 >= 7 → TRUE
  • BƯỚC 4: Dừng kiểm tra, thực thi khối lệnh tương ứng
  • BƯỚC 5: grade = "Khá", in "Tốt! Bạn đạt loại: Khá"
  • BƯỚC 6: Tiếp tục chương trình → in "Kết thúc xếp loại"

Tốt! Bạn đạt loại: Khá

TOÁN TỬ BA NGÔI

1.3 Toán tử ba ngôi (Ternary Operator)

Định nghĩa: Là cú pháp rút gọn của if...else, dùng khi cần lựa chọn giữa hai giá trị.

Cú pháp
điều_kiện ? giá_trị_nếu_đúng : giá_trị_nếu_sai;

Ý nghĩa:

Giúp viết mã ngắn gọn và súc tích hơn, nhất là khi chỉ cần trả về một giá trị.

TernaryOperator.java
public class TernaryOperator {
    public static void main(String[] args) {
        double score = 6.0;
        
        // Sử dụng if-else thông thường
        String result1;
        if (score >= 5) {
            result1 = "Đậu";
        } else {
            result1 = "Rớt";
        }
        
        // Sử dụng toán tử ba ngôi
        String result2 = (score >= 5) ? "Đậu" : "Rớt";
        
        System.out.println("Kết quả (if-else): " + result1);
        System.out.println("Kết quả (ternary): " + result2);
        
        // Ví dụ phức tạp hơn
        int number = 10;
        String message = (number % 2 == 0) ? "Số chẵn" : "Số lẻ";
        System.out.println(number + " là " + message);
    }
}

CÂU LỆNH SWITCH-CASE

1.4 Câu lệnh switch...case

Định nghĩa: Câu lệnh switch dùng để so sánh một giá trị với nhiều trường hợp khác nhau (case).

Cú pháp
switch (biểu_thức) {
    case giá_trị_1:
        <khối_lệnh_1>;
        break;
    case giá_trị_2:
        <khối_lệnh_2>;
        break;
    ...
    case giá_trị_n:
        <khối_lệnh_n>;
        break;
    default:
        <khối_lệnh_mặc_định>;
}

Lưu đồ thuật toán câu lệnh IF

SwitchExample.java
public class SwitchExample {
    public static void main(String[] args) {
        int dayOfWeek = 3;
        String dayName;
        
        switch (dayOfWeek) {
            case 1:
                dayName = "Chủ Nhật";
                break;
            case 2:
                dayName = "Thứ Hai";
                break;
            case 3:
                dayName = "Thứ Ba";
                break;
            case 4:
                dayName = "Thứ Tư";
                break;
            case 5:
                dayName = "Thứ Năm";
                break;
            case 6:
                dayName = "Thứ Sáu";
                break;
            case 7:
                dayName = "Thứ Bảy";
                break;
            default:
                dayName = "Không hợp lệ";
        }
        
        System.out.println("Ngày " + dayOfWeek + " là: " + dayName);
    }
}

Giải thích chi tiết:

  • BƯỚC 1: dayOfWeek = 3
  • BƯỚC 2: So sánh với case 1 → không khớp
  • BƯỚC 3: So sánh với case 2 → không khớp
  • BƯỚC 4: So sánh với case 3 → KHỚP
  • BƯỚC 5: Thực thi dayName = "Thứ Ba"
  • BƯỚC 6: Gặp break → thoát switch
  • BƯỚC 7: In kết quả

Kết quả:
Ngày 3 là: Thứ Ba

TOÁN TỬ LOGIC

1.5 Toán tử logic &&, ||, !

Định nghĩa:

  • && (AND): Đúng khi cả hai điều kiện đúng
  • || (OR): Đúng khi ít nhất một điều kiện đúng
  • ! (NOT): Phủ định điều kiện (đúng thành sai và ngược lại)
Nội dung AND (&&) OR (||)
Lưu đồ
Mô tả Chỉ trả về TRUE khi cả hai điều kiện đều đúng Trả về TRUE khi ít nhất một điều kiện đúng
Short-circuit Evaluation Nếu điều kiện đầu tiên sai → kết thúc ngay Nếu điều kiện đầu tiên đúng → kết thúc ngay

Ý nghĩa:

Toán tử && và || thực hiện đánh giá từ trái sang phải và có cơ chế "short-circuit" (ngắt mạch), nghĩa là dừng kiểm tra ngay khi có kết quả xác định.

LogicalOperators.java
public class LogicalOperators {
    public static void main(String[] args) {
        int age = 20;
        boolean hasID = true;
        
        // Sử dụng AND (&&)
        if (age >= 18 && hasID) {
            System.out.println("Được vào cửa");
        } else {
            System.out.println("Không được vào");
        }
        
        // Sử dụng OR (||)
        boolean isWeekend = true;
        boolean isHoliday = false;
        
        if (isWeekend || isHoliday) {
            System.out.println("Hôm nay được nghỉ");
        } else {
            System.out.println("Hôm nay đi làm");
        }
        
        // Sử dụng NOT (!)
        boolean isRaining = false;
        
        if (!isRaining) {
            System.out.println("Trời không mưa, có thể đi chơi");
        } else {
            System.out.println("Trời mưa, ở nhà");
        }
    }
}

MẸO VÀ THỦ THUẬT

1.6 Mẹo và cú pháp viết nhanh

Kỹ thuật Mô tả Ví dụ
if (a) Kiểm tra nếu a tồn tại (truthy) if (name != null)
if (!a) Kiểm tra nếu a không tồn tại (falsy) if (!isEmpty)
Viết gọn if đơn Bỏ {} nếu chỉ có 1 lệnh if (condition) statement;
Toán tử so sánh So sánh chuỗi dùng equals() if (str.equals("hello"))
QuickSyntax.java
public class QuickSyntax {
    public static void main(String[] args) {
        String name = "John";
        boolean isEmpty = false;
        
        // Kiểm tra null và rỗng
        if (name != null && !name.isEmpty()) {
            System.out.println("Tên hợp lệ: " + name);
        }
        
        // Viết gọn if đơn
        int x = 10;
        if (x > 5) System.out.println("x lớn hơn 5");
        
        // So sánh chuỗi
        String input = "hello";
        if (input.equals("hello")) {
            System.out.println("Chào mừng!");
        }
    }
}

Best Practices:

  • Luôn sử dụng {} cho khối lệnh, kể cả khi chỉ có 1 dòng
  • Sử dụng equals() để so sánh chuỗi, không dùng ==
  • Kiểm tra null trước khi gọi phương thức trên đối tượng
  • Sử dụng toán tử ba ngôi cho các trường hợp đơn giản
