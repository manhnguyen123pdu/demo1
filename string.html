CHƯƠNG 1: TỔNG QUAN VỀ CHUỖI TRONG JAVA

1.1. Định nghĩa

Chuỗi (String) trong Java là một đối tượng đại diện cho một chuỗi các ký tự (sequence of characters). Lớp String thuộc gói java.lang và được sử dụng để tạo, thao tác và xử lý các chuỗi ký tự.

Khái niệm cơ bản
// Chuỗi là đối tượng trong Java
String message = "Hello Java";
System.out.println(message); // Output: Hello Java

1.2. Đặc điểm quan trọng

Immutable (Bất biến)

Đối tượng String không thể thay đổi sau khi tạo. Mọi thao tác tạo ra chuỗi mới.

String Pool

Cơ chế lưu trữ đặc biệt trong bộ nhớ Heap để tối ưu bộ nhớ.

Thread-safe

An toàn trong môi trường đa luồng do tính bất biến.

Tại sao String là Immutable?

  • Bảo mật: Ngăn chặn thay đổi giá trị chuỗi quan trọng
  • Thread-safe: Có thể chia sẻ an toàn giữa các thread
  • Tối ưu bộ nhớ: String Pool hoạt động hiệu quả
  • Hiệu suất: Bộ nhớ cache hashcode

Lưu ý quan trọng

String là immutable, nhưng biến tham chiếu đến String có thể thay đổi. Điều này có thể gây nhầm lẫn cho người mới học.

So sánh String Literal và new String()

Tiêu chí String Literal new String()
Bộ nhớ String Pool Heap Memory
Hiệu suất Tốt hơn (tái sử dụng) Kém hơn (tạo mới)
Số lượng object 1 (nếu cùng giá trị) Nhiều object
So sánh == true (cùng tham chiếu) false (khác tham chiếu)
So sánh equals() true (cùng nội dung) true (cùng nội dung)

Ví dụ charAt()
public class ViDuCharAt {
    public static void main(String[] args) {
        String str = "Java Programming";

        // Lấy ký tự tại các vị trí khác nhau
        System.out.println("Ký tự tại vị trí 0: " + str.charAt(0)); // J
        System.out.println("Ký tự tại vị trí 5: " + str.charAt(5)); // P
        System.out.println("Ký tự tại vị trí 10: " + str.charAt(10)); // m

        // Duyệt qua tất cả ký tự trong chuỗi
        System.out.println("\\nDuyệt từ đầu đến cuối:");
        for (int i = 0; i < str.length(); i++) {
            System.out.println("Vị trí " + i + ": " + str.charAt(i));
        }

        // Duyệt ngược từ cuối lên đầu
        System.out.println("\\nDuyệt từ cuối lên đầu:");
        for (int i = str.length() - 1; i >= 0; i--) {
            System.out.println("Vị trí " + i + ": " + str.charAt(i));
        }

        // Ứng dụng: Đếm số lần xuất hiện của một ký tự
        char timKyTu = 'a';
        int dem = 0;
        for (int i = 0; i < str.length(); i++) {
            if (str.charAt(i) == timKyTu) {
                dem++;
            }
        }
        System.out.println("\\nKý tự '" + timKyTu + "' xuất hiện " + dem + " lần");

        // Xử lý ngoại lệ
        try {
            System.out.println("Thử lấy vị trí 100: " + str.charAt(100));
        } catch (StringIndexOutOfBoundsException e) {
            System.out.println("Lỗi: Vị trí không hợp lệ!");
        }
    }
}

public class ViDuIndexOf {
    public static void main(String[] args) {

3.8. Phương thức split()

Tách chuỗi thành mảng các chuỗi con dựa trên biểu thức chính quy (regex).

Cú pháp:
String[] mangChuoi = chuoi.split(regex);
String[] mangChuoi = chuoi.split(regex, gioiHan);
Ví dụ split()
public class ViDuSplit {
    public static void main(String[] args) {
        // Tách theo dấu phẩy
        String str1 = "apple,banana,orange,grape";
        String[] fruits = str1.split(",");
        System.out.println("Tách theo dấu phẩy:");
        for (String fruit : fruits) {
            System.out.println("- " + fruit);
        }

        // Tách theo khoảng trắng
        String str2 = "Java is a programming language";
        String[] words = str2.split(" ");
        System.out.println("\\nTách theo khoảng trắng:");
        for (String word : words) {
            System.out.println("- " + word);
        }

        // Tách với giới hạn số phần tử
        String str3 = "one:two:three:four:five";
        String[] parts = str3.split(":", 3);
        System.out.println("\\nTách với giới hạn 3 phần:");
        for (String part : parts) {
            System.out.println("- " + part);
        }

        // Tách theo nhiều ký tự
        String str4 = "apple, banana; orange.grape";
        String[] fruits2 = str4.split("[,;. ]+");
        System.out.println("\\nTách theo nhiều ký tự phân cách:");
        for (String fruit : fruits2) {
            System.out.println("- " + fruit);
        }

        // Ứng dụng: Phân tích CSV
        String csv = "John,Doe,30,Developer";
        String[] data = csv.split(",");
        System.out.println("\\nPhân tích CSV:");
        System.out.println("First Name: " + data[0]);
        System.out.println("Last Name: " + data[1]);
        System.out.println("Age: " + data[2]);
        System.out.println("Job: " + data[3]);

        // Ứng dụng: Tách đường dẫn
        String path = "/usr/local/bin/java";
        String[] directories = path.split("/");
        System.out.println("\\nCác thư mục trong đường dẫn:");
        for (String dir : directories) {
            if (!dir.isEmpty()) {
                System.out.println("- " + dir);
            }
        }

        // Tách từng ký tự
        String str5 = "Hello";
        String[] chars = str5.split("");
        System.out.println("\\nTách từng ký tự:");
        for (String ch : chars) {
            System.out.print(ch + " ");
        }

        // Regex đặc biệt: dấu chấm
        String str6 = "a.b.c";
        String[] parts2 = str6.split("\\\\.");
        System.out.println("\\n\\nTách theo dấu chấm:");
        for (String part : parts2) {
            System.out.println("- " + part);
        }
    }
}

3.9. Phương thức toUpperCase() và toLowerCase()

Chuyển đổi tất cả ký tự trong chuỗi thành chữ hoa (toUpperCase) hoặc chữ thường (toLowerCase).

Cú pháp:
String chuoiHoa = chuoi.toUpperCase();
String chuoiThuong = chuoi.toLowerCase();
Ví dụ toUpperCase() & toLowerCase()
public class ViDuChuHoaThuong {
    public static void main(String[] args) {

        String original = "Java Programming Language";

        // Chuyển thành chữ hoa
        String upper = original.toUpperCase();
        System.out.println("Chữ hoa: " + upper); // JAVA PROGRAMMING LANGUAGE

        // Chuyển thành chữ thường
        String lower = original.toLowerCase();
        System.out.println("Chữ thường: " + lower); // java programming language

        // Ứng dụng: So sánh không phân biệt hoa thường
        String input1 = "JAVA";
        String input2 = "java";
        System.out.println("\\nSo sánh phân biệt hoa thường: " + input1.equals(input2)); // false

        System.out.println("So sánh không phân biệt: " + input1.equalsIgnoreCase(input2)); // true

        System.out.println("So sánh qua chuyển đổi: " + input1.toLowerCase().equals(input2.toLowerCase())); // true

        // Ứng dụng: Chuẩn hóa dữ liệu nhập
        String userInput = " ADMIN ";
        String normalized = userInput.trim().toLowerCase();
        System.out.println("\\nDữ liệu chuẩn hóa: '" + normalized + "'");

        if (normalized.equals("admin")) {
            System.out.println("Đăng nhập với quyền admin");
        }

        // Ứng dụng: Tạo mã hóa đơn
        String customerName = "Nguyễn Văn A";
        String date = "20240315";
        String invoiceCode = customerName.toUpperCase().replace(" ", "") + date;
        System.out.println("Mã hóa đơn: " + invoiceCode);

        // Ứng dụng: Kiểm tra mật khẩu có chữ hoa và chữ thường
        String password = "Pass123";
        boolean hasUpperCase = !password.equals(password.toLowerCase());
        boolean hasLowerCase = !password.equals(password.toUpperCase());

        System.out.println("\\nKiểm tra mật khẩu:");
        System.out.println("Có chữ hoa: " + hasUpperCase);
        System.out.println("Có chữ thường: " + hasLowerCase);

        // Chuẩn hóa chữ cái đầu mỗi từ
        String name = "john doe";
        String[] nameParts = name.split(" ");
        StringBuilder capitalized = new StringBuilder();

        for (String part : nameParts) {
            if (!part.isEmpty()) {
                capitalized.append(Character.toUpperCase(part.charAt(0)))
                    .append(part.substring(1).toLowerCase())
                    .append(" ");
            }
        }

        System.out.println("Tên đã chuẩn hóa: " + capitalized.toString().trim());

    }
}

2. Phương thức compareTo()

2.1. Cú pháp
Cú pháp: int result = str1.compareTo(str2);
2.2. Nguyên tắc so sánh

Java so sánh từng ký tự một, dựa trên mã Unicode (ASCII mở rộng):

Kết quả trả về Ý nghĩa Ví dụ

< 0 (số âm) str1 nhỏ hơn str2 "Apple" < "Banana"
> 0 (số dương) str1 lớn hơn str2 "Zebra" > "Apple"

Lưu ý: So sánh phân biệt CHỮ HOA - chữ thường

2.3. Ví dụ 1: Chuỗi giống nhau

Ví dụ compareTo() - Khác chữ hoa thường

Giải thích: 'J' (74) < 'j' (106)

Quy tắc nhớ kết quả compareTo()

Số -32 là hiệu mã Unicode, không cần nhớ chính xác, chỉ cần nhớ:

  • Âm (negative) → str1 nhỏ hơn str2
  • 0 → hai chuỗi bằng nhau
  • Dương (positive) → str1 lớn hơn str2
2.5. Ví dụ 3: So sánh nhiều trường hợp

        // join() với mảng
        String[] fruits = {"Apple", "Banana", "Orange"};
        String joined2 = String.join(" | ", fruits);
        System.out.println("Trái cây: " + joined2);
    }
}

Thử nghiệm phương thức join()

Kết quả:

Ứng dụng thực tế của join():

  • Tạo đường dẫn file: String.join("/", "home", "user", "file.txt")
  • Tạo CSV: String.join(",", name, age, address)
  • Tạo URL query: String.join("&", "q=java", "page=1", "sort=asc")
  • Tạo danh sách HTML: "<li>" + String.join("</li><li>", items) + "</li>"

5.4. Phương thức format()

1. Khái niệm

String.format() là phương thức tĩnh của lớp String, dùng để:

  • Tạo chuỗi có định dạng
  • Chèn dữ liệu vào chuỗi theo mẫu (template)
  • Giúp code dễ đọc, dễ bảo trì hơn so với nối chuỗi +

➡ Tương tự hàm printf() trong C/C++.

2. Cú pháp tổng quát

String result = String.format(formatString, arg1, arg2, ...);

3. Các ký hiệu định dạng thường dùng

Ký hiệu Kiểu dữ liệu Ví dụ Kết quả
%s Chuỗi (String) "Java" Java
%d Số nguyên 10 10
%f Số thực 3.14159 3.141590
%.2f Số thực 2 chữ số 3.14159 3.14
%c Ký tự 'A' A
%b Boolean true true
%n Xuống dòng %n \n

4. Ví dụ cơ bản

Ví dụ format() cơ bản
// Định dạng đơn giản
String s = String.format("Hello, %s!", "World");
System.out.println(s); // Hello, World!

// Với số nguyên
int age = 20;
String s1 = String.format("Tuổi của bạn là %d", age);
System.out.println(s1); // Tuổi của bạn là 20

// Với số thực
double score = 8.56789;
String s2 = String.format("Điểm: %.2f", score);
System.out.println(s2); // Điểm: 8.57

5. Chèn nhiều giá trị

Chèn nhiều giá trị
String name = "An";
int age = 21;
double gpa = 3.75;

String info = String.format(
    "Tên: %s | Tuổi: %d | GPA: %.2f",
    name, age, gpa
);

System.out.println(info); // Tên: An | Tuổi: 21 | GPA: 3.75

6. Định dạng độ rộng (canh lề)

Định dạng độ rộng
// Canh phải (width 10)
String s1 = String.format("|%10s|", "Java");
System.out.println(s1); // | Java|

// Canh trái (width 10)
String s2 = String.format("|%-10s|", "Java");
System.out.println(s2); // |Java |

// Định dạng số với độ rộng
int num = 123;
String s3 = String.format("|%8d|", num);
System.out.println(s3); // | 123|

// Định dạng số thực với độ rộng và số chữ số
double price = 19.99;
String s4 = String.format("|%10.2f|", price);
System.out.println(s4); // | 19.99|

Thử nghiệm phương thức format()

Tổng hợp các phương pháp nối chuỗi

Phương pháp Ưu điểm Nhược điểm Khi nào dùng
Toán tử + Đơn giản, tự động chuyển đổi Hiệu năng kém trong vòng lặp Nối ít, đơn giản
concat() Tốc độ nhanh cho nối đơn Chỉ nhận String, không chain nhiều Nối 2 chuỗi cố định
join() Hiệu quả cho danh sách Chỉ cho mảng/collection Nối nhiều chuỗi có phân cách
format() Định dạng phức tạp, dễ đọc Hiệu năng chậm hơn Tạo chuỗi có định dạng
StringBuilder Hiệu năng cao trong vòng lặp Cú pháp dài hơn Nối nhiều trong vòng lặp
