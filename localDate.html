PHẦN MỞ ĐẦU

1. GIỚI THIỆU TỔNG QUAN

1.1 Vấn đề với Date/Time cũ (trước Java 8)

Các lớp java.util.Date, java.util.Calendar, java.text.SimpleDateFormat khó sử dụng

Không thread-safe: SimpleDateFormat không an toàn trong môi trường đa luồng

API không nhất quán: Phương thức đặt tên khó hiểu, dễ nhầm lẫn

Xử lý timezone phức tạp: Thiếu hỗ trợ cho các múi giờ

Không tuân thủ chuẩn ISO-8601

Vấn đề với Date/Time cũ
// Ví dụ cũ - PROBLEMATIC
Date date = new Date(); // Mutable, không thread-safe
Calendar cal = Calendar.getInstance();
cal.set(2024, Calendar.DECEMBER, 25); // Tháng bắt đầu từ 0!
SimpleDateFormat sdf = new SimpleDateFormat("dd/MM/yyyy"); // Not thread-safe
Vấn đề chính:
  • Thread-safety: SimpleDateFormat không an toàn trong đa luồng
  • API thiết kế kém: Tháng bắt đầu từ 0, năm từ 1900
  • Mutable objects: Có thể thay đổi trạng thái sau khi tạo
  • Timezone issues: Xử lý timezone phức tạp và dễ sai

1.2 Giới thiệu java.time API (Java 8+)

Package java.time (JSR-310): Thiết kế dựa trên thư viện Joda-Time

Đặc điểm nổi bật: Thread-safe, immutable, API rõ ràng

Tuân thủ ISO-8601: Chuẩn quốc tế về biểu diễn ngày giờ

Phân chia rõ ràng: Mỗi lớp phục vụ một mục đích cụ thể

Tích hợp tốt với Java Stream API

Ví dụ với java.time API
// Ví dụ mới - BETTER
LocalDate date = LocalDate.of(2024, 12, 25); // Immutable, thread-safe
Ưu điểm của java.time:
  • Immutability: Tất cả các lớp đều immutable → thread-safe
  • API nhất quán: Phương thức đặt tên rõ ràng, dễ hiểu
  • ISO-8601 compliant: Tuân theo chuẩn quốc tế
  • Phân chia rõ ràng: Mỗi lớp có mục đích cụ thể
  • Timezone hỗ trợ tốt: Xử lý timezone dễ dàng

1.3 CẤU TRÚC PACKAGE

java.time ← Core classes
├── LocalDate ← Chỉ ngày (yyyy-MM-dd)
├── LocalTime ← Chỉ giờ (HH:mm:ss)
├── LocalDateTime ← Ngày + giờ (không timezone)
├── ZonedDateTime ← Ngày + giờ + timezone
├── Instant ← Thời điểm trên timeline
├── Period ← Khoảng ngày (năm, tháng, ngày)
├── Duration ← Khoảng thời gian (giờ, phút, giây)
└── ...

java.time.format ← Định dạng và phân tích
├── DateTimeFormatter ← Định dạng ngày giờ
└── ...

java.time.temporal ← Truy cập trường và đơn vị
├── TemporalAccessor ← Interface cơ bản
├── TemporalAdjusters ← Điều chỉnh ngày
└── ...

java.time.zone ← Xử lý múi giờ
└── ZoneId, ZoneOffset ← Múi giờ và độ lệch
Phân tích cấu trúc package:

java.time: Package chính chứa các lớp cốt lõi để làm việc với ngày giờ.

java.time.format: Cung cấp các lớp để định dạng và phân tích ngày giờ.

java.time.temporal: Cung cấp các interface và lớp để truy cập các trường ngày giờ.

java.time.zone: Hỗ trợ làm việc với các múi giờ.

PHẦN 2: LOCALDATE - CHỈ NGÀY THÁNG NĂM

2.1 KHÁI NIỆM

2.1.1 Định nghĩa

LocalDate đại diện cho một ngày cụ thể (năm-tháng-ngày)

Không chứa thông tin về thời gian và múi giờ

Là immutable class: Một khi tạo ra không thể thay đổi

Thread-safe: An toàn trong môi trường đa luồng

2.1.2 Đặc điểm

Chỉ chứa: Năm, tháng, ngày

Định dạng mặc định: yyyy-MM-dd (ISO-8601)

Không ảnh hưởng bởi timezone

Phù hợp cho: Ngày sinh, ngày sự kiện, ngày hẹn...

Phân tích về LocalDate:

Mô tả: Đại diện cho ngày (năm-tháng-ngày) không có thời gian và timezone

Phân tích: Immutable, thread-safe, định dạng ISO-8601 (yyyy-MM-dd)

Ứng dụng: Thích hợp cho các trường hợp chỉ cần thông tin ngày, không cần giờ và timezone như ngày sinh, ngày ký hợp đồng, ngày sự kiện...

2.2 CÚ PHÁP KHAI BÁO

2.2.1 Tạo đối tượng LocalDate

Các cách tạo LocalDate
// Cách 1: Từ ngày hiện tại
LocalDate today = LocalDate.now();

// Cách 2: Từ các thành phần
LocalDate specificDate = LocalDate.of(2024, 12, 25);
LocalDate dateWithMonthEnum = LocalDate.of(2024, Month.DECEMBER, 25);

// Cách 3: Parse từ chuỗi ISO
LocalDate parsedDate = LocalDate.parse("2024-12-25");

// Cách 4: Từ epoch day
LocalDate fromEpoch = LocalDate.ofEpochDay(365 * 50); // 50 năm từ 1970

// Cách 5: Từ năm và ngày trong năm
LocalDate fromDayOfYear = LocalDate.ofYearDay(2024, 200); // Ngày thứ 200 của năm 2024

2.2 PHƯƠNG THỨC TẠO LOCALDATE

2.2.1 Phương thức now()

Mô tả: Tạo LocalDate từ ngày hiện tại của hệ thống

Cú pháp:

Cú pháp now()
static LocalDate now()
static LocalDate now(Clock clock)
static LocalDate now(ZoneId zone)

Ví dụ:

Ví dụ now()
LocalDate today = LocalDate.now();
Phân tích:

Lấy ngày từ đồng hồ hệ thống, phụ thuộc vào default timezone

Phiên bản không tham số sử dụng system clock và default timezone

Phiên bản có tham số cho phép kiểm soát clock và timezone

2.2.2 Phương thức of()

Mô tả: Tạo LocalDate từ các thành phần năm, tháng, ngày

Cú pháp:

Cú pháp of()
static LocalDate of(int year, int month, int dayOfMonth)
static LocalDate of(int year, Month month, int dayOfMonth)

Ví dụ:

Ví dụ of()
LocalDate date1 = LocalDate.of(2024, 12, 25);
LocalDate date2 = LocalDate.of(2024, Month.DECEMBER, 25);
Phân tích:

Kiểm tra tính hợp lệ, throw DateTimeException nếu không hợp lệ

Tháng có thể là số (1-12) hoặc enum Month

Ngày phải hợp lệ với tháng (ví dụ: không thể có ngày 31 trong tháng 2)

2.2.3 Phương thức parse()

Mô tả: Parse LocalDate từ chuỗi theo định dạng ISO

Cú pháp:

Cú pháp parse()
static LocalDate parse(CharSequence text)
static LocalDate parse(CharSequence text, DateTimeFormatter formatter)

Ví dụ:

Ví dụ parse()
LocalDate date1 = LocalDate.parse("2024-12-25");
LocalDate date2 = LocalDate.parse("25/12/2024",
    DateTimeFormatter.ofPattern("dd/MM/yyyy"));
Phân tích:

Mặc định định dạng yyyy-MM-dd, có thể custom với formatter

Ném DateTimeParseException nếu chuỗi không hợp lệ

Hữu ích khi đọc dữ liệu từ file, database hoặc API

Thử nghiệm tạo LocalDate

Giúp code dễ đọc và type-safe hơn so với dùng số

Phù hợp khi cần giá trị số để tính toán

Ít gây nhầm lẫn hơn so với Calendar (tháng bắt đầu từ 0)

Ví dụ getDayOfMonth()

Phân tích:

Trả về int từ 1 đến 31 (tùy tháng)

Không giống như Calendar trả về ngày từ 0

Luôn trả về ngày hợp lệ trong tháng

2.3.5 Phương thức getDayOfWeek()

Mô tả: Lấy thứ trong tuần dưới dạng DayOfWeek enum

Cú pháp: DayOfWeek getDayOfWeek()

Ví dụ:

Ví dụ getDayOfWeek()

Phân tích:

Trả về enum từ MONDAY đến SUNDAY

Tuân theo tiêu chuẩn ISO-8601 (thứ 2 là ngày đầu tuần)

Không giống như Calendar (SUNDAY = 1, MONDAY = 2, ...)

2.3.6 Phương thức getDayOfYear()

Mô tả: Lấy ngày thứ bao nhiêu trong năm

Cú pháp: int getDayOfYear()

Ví dụ:

Ví dụ getDayOfYear()
LocalDate date = LocalDate.of(2024, 12, 31);
int dayOfYear = date.getDayOfYear(); // 366 (năm nhuận)
Phân tích:

Trả về int từ 1 đến 365 (hoặc 366 nếu năm nhuận)

Hữu ích cho các tính toán liên quan đến ngày trong năm

Tự động tính năm nhuận

2.3.7 Phương thức isLeapYear()

Mô tả: Kiểm tra có phải năm nhuận không

Cú pháp: boolean isLeapYear()

Ví dụ:

Ví dụ isLeapYear()
LocalDate date1 = LocalDate.of(2024, 12, 25);
boolean leap1 = date1.isLeapYear(); // true

LocalDate date2 = LocalDate.of(2023, 12, 25);
boolean leap2 = date2.isLeapYear(); // false
Phân tích:

Năm nhuận chia hết cho 4, nhưng không chia hết cho 100 trừ khi chia hết cho 400

Công thức: (năm % 4 == 0) && (năm % 100 != 0 || năm % 400 == 0)

Ví dụ: 2000 là năm nhuận, 1900 không phải năm nhuận

2.3.8 Phương thức lengthOfMonth()

Mô tả: Lấy số ngày trong tháng

Cú pháp: int lengthOfMonth()

Ví dụ:

Ví dụ lengthOfMonth()
LocalDate feb2024 = LocalDate.of(2024, 2, 1);
int days = feb2024.lengthOfMonth(); // 29 (năm nhuận)

LocalDate feb2023 = LocalDate.of(2023, 2, 1);
int days = feb2023.lengthOfMonth(); // 28
Phân tích:

Tự động tính năm nhuận, tháng có 28, 29, 30 hoặc 31 ngày

Hữu ích khi cần biết số ngày trong tháng để tính toán

Trả về chính xác số ngày cho tháng hiện tại

2.3.9 Phương thức lengthOfYear()

Cú pháp: int lengthOfYear()

Ví dụ:

Ví dụ lengthOfYear()

int days = date2024.lengthOfYear(); // 366

LocalDate date2023 = LocalDate.of(2023, 1, 1);

Phân tích:

Trả về 365 cho năm thường, 366 cho năm nhuận

Hữu ích cho các tính toán liên quan đến tổng số ngày trong năm

Tự động xác định năm nhuận

2.4 PHƯƠNG THỨC THAO TÁC (IMMUTABLE)

2.4.1 Phương thức plusDays()

Mô tả: Tạo LocalDate mới bằng cách cộng thêm số ngày

Cú pháp: LocalDate plusDays(long daysToAdd)

Ví dụ:

Ví dụ plusDays()

LocalDate yesterday = today.plusDays(-1); // 2024-12-24
Phân tích:

Trả về đối tượng mới, không thay đổi đối tượng gốc (immutable)

Có thể dùng số âm để trừ ngày

Tự động điều chỉnh tháng/năm nếu cần

2.4.2 Phương thức plusWeeks()

Mô tả: Tạo LocalDate mới bằng cách cộng thêm số tuần

Cú pháp: LocalDate plusWeeks(long weeksToAdd)

Ví dụ:

Ví dụ plusWeeks()
LocalDate today = LocalDate.of(2024, 12, 25);
LocalDate nextWeek = today.plusWeeks(1); // 2025-01-01
Phân tích:

1 tuần = 7 ngày, tự động điều chỉnh tháng/năm nếu cần

Hữu ích cho các tính toán theo tuần

Giữ nguyên immutable property

2.4.3 Phương thức plusMonths()

Mô tả: Tạo LocalDate mới bằng cách cộng thêm số tháng

Cú pháp: LocalDate plusMonths(long monthsToAdd)

Ví dụ:

Ví dụ plusMonths()

Giữ nguyên ngày, nếu ngày không hợp lệ thì lấy ngày cuối cùng của tháng

Ví dụ: 31/1 + 1 tháng = 29/2 (năm nhuận) hoặc 28/2

Hữu ích cho các tính toán về thời hạn, kỳ hạn

2.4.4 Phương thức plusYears()

Tự động xử lý năm nhuận, nếu 29/2 không hợp lệ thì lấy 28/2

Cú pháp: LocalDate with(TemporalAdjuster adjuster)

Ví dụ:

Ví dụ với TemporalAdjuster
LocalDate date = LocalDate.of(2024, 12, 25);

// Ngày đầu tháng
LocalDate firstDay = date.with(TemporalAdjusters.firstDayOfMonth());

// Ngày cuối tháng
LocalDate lastDay = date.with(TemporalAdjusters.lastDayOfMonth());

// Thứ Hai tiếp theo
LocalDate nextMonday = date.with(TemporalAdjusters.next(DayOfWeek.MONDAY));
Phân tích:

TemporalAdjuster cung cấp các điều chỉnh phức tạp

Có sẵn các adjuster phổ biến trong TemporalAdjusters class

Có thể tạo custom adjuster bằng cách implement TemporalAdjuster interface

Thử nghiệm điều chỉnh

Bài tập 1: Tính tuổi từ ngày sinh

CalculateAge.java
import java.time.LocalDate;
import java.time.Period;

public class CalculateAge {
    public static void main(String[] args) {
        LocalDate birthDate = LocalDate.of(1990, 5, 15);
        LocalDate today = LocalDate.now();

        Period age = Period.between(birthDate, today);

        System.out.println("Ngày sinh: " + birthDate);
        System.out.println("Hôm nay: " + today);
        System.out.println("Tuổi: " + age.getYears() + " năm " +
            age.getMonths() + " tháng " +
            age.getDays() + " ngày");
    }
}

Bài tập 2: Tính ngày đến hạn (30 ngày sau)

DueDateCalculator.java
import java.time.LocalDate;

public class DueDateCalculator {
    public static void main(String[] args) {
        LocalDate startDate = LocalDate.of(2024, 1, 15);
        LocalDate dueDate = startDate.plusDays(30);

        System.out.println("Ngày bắt đầu: " + startDate);
        System.out.println("Ngày đến hạn (30 ngày sau): " + dueDate);
        
        // Kiểm tra nếu quá hạn
        LocalDate today = LocalDate.now();
        if (today.isAfter(dueDate)) {
            System.out.println("Đã quá hạn!");
        } else {
            System.out.println("Còn trong hạn");
        }
    }
}

Bài tập 3: Kiểm tra năm nhuận

LeapYearChecker.java
import java.time.LocalDate;

public class LeapYearChecker {
    public static void main(String[] args) {
        int[] years = {2000, 1900, 2024, 2023, 2020};

        for (int year : years) {
            LocalDate date = LocalDate.of(year, 1, 1);
            boolean isLeap = date.isLeapYear();
            
            System.out.println("Năm " + year + " là năm nhuận? " + isLeap);
        }
    }
}
