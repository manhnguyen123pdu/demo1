VÒNG LẶP (LOOPS) TRONG JAVA

Hướng dẫn toàn diện về các cấu trúc vòng lặp trong Java

2.1 Khái niệm và mục đích của vòng lặp

Định nghĩa:

Vòng lặp là cấu trúc điều khiển cho phép thực thi một khối lệnh nhiều lần dựa trên một điều kiện nhất định.

Mục đích của vòng lặp:

  • Tự động hóa: Thực hiện các tác vụ lặp đi lặp lại
  • Xử lý tập hợp: Duyệt qua mảng, danh sách, collection
  • Tính toán lặp: Tính giai thừa, tổng dãy số,...
  • Kiểm tra liên tục: Đợi sự kiện, kiểm tra trạng thái

Các loại vòng lặp trong Java:

  • for - Biết trước số lần lặp
  • while - Không biết trước số lần lặp, kiểm tra điều kiện trước
  • do-while - Không biết trước số lần lặp, thực hiện ít nhất 1 lần
  • for-each - Duyệt qua mảng và collection

2.2 Vòng lặp for

Định nghĩa:

Vòng lặp for được sử dụng khi biết trước số lần lặp.

Lưu đồ thuật toán:

BẮT ĐẦU
↓
Khởi tạo biến đếm
↓
Kiểm tra điều kiện
↓
Điều kiện ĐÚNG
↓
Thực thi khối lệnh
↓
Cập nhật biến đếm
Điều kiện SAI
↓
KẾT THÚC

Lưu đồ mô tả hoạt động của vòng lặp for

Cú pháp:

for (khởi_tạo; điều_kiện; cập_nhật) {
    // Khối lệnh thực thi
}

Giải thích các thành phần:

  • khởi_tạo: Khai báo và khởi tạo biến đếm (chạy 1 lần duy nhất)
  • điều_kiện: Kiểm tra trước mỗi lần lặp, nếu true thì tiếp tục
  • cập_nhật: Thay đổi biến đếm sau mỗi lần lặp
  • khối lệnh: Các câu lệnh được thực thi trong mỗi vòng lặp

Ví dụ 1: In các số từ 1 đến 5

public class ForLoopBasic {
    public static void main(String[] args) {
        System.out.println("=== VÒNG LẶP FOR CƠ BẢN ===");
        
        for (int i = 1; i <= 5; i++) {
            System.out.println("Lần lặp thứ: " + i);
        } 
        System.out.println("Kết thúc vòng lặp");
    }
}

Giải thích chi tiết luồng thực thi:

  1. BƯỚC 1: Khởi tạo - int i = 1 (chạy 1 lần)
  2. BƯỚC 2: Kiểm tra điều kiện - i <= 5 → 1 <= 5 → TRUE
  3. BƯỚC 3: Thực thi khối lệnh - in "Lần lặp thứ: 1"
  4. BƯỚC 4: Cập nhật - i++ → i = 2
  5. BƯỚC 5: Kiểm tra điều kiện - i <= 5 → 2 <= 5 → TRUE
  6. BƯỚC 6: Thực thi khối lệnh - in "Lần lặp thứ: 2"
  7. BƯỚC 7: Cập nhật - i++ → i = 3
  8. ...
  9. BƯỚC n: Khi i = 6 → điều kiện 6 <= 5 → FALSE → thoát vòng lặp

Kết quả:

=== VÒNG LẶP FOR CƠ BẢN ===
Lần lặp thứ: 1
Lần lặp thứ: 2
Lần lặp thứ: 3
Lần lặp thứ: 4
Lần lặp thứ: 5
Kết thúc vòng lặp

Ví dụ 2: Tính tổng các số từ 1 đến 10

public class ForLoopSum {
    public static void main(String[] args) {
        System.out.println("=== TÍNH TỔNG DÃY SỐ ===");
        
        int sum = 0;
        
        for (int i = 1; i <= 10; i++) {
            sum = sum + i;  // Hoặc sum += i;
            System.out.println("Số hiện tại: " + i + " - Tổng tạm thời: " + sum);
        }
        
        System.out.println("Tổng cuối cùng từ 1 đến 10 là: " + sum);
    }
}

Giải thích chi tiết:

  • VÒNG LẶP 1: i = 1, sum = 0 + 1 = 1
  • VÒNG LẶP 2: i = 2, sum = 1 + 2 = 3
  • VÒNG LẶP 3: i = 3, sum = 3 + 3 = 6
  • VÒNG LẶP 4: i = 4, sum = 6 + 4 = 10
  • ...
  • VÒNG LẶP 10: i = 10, sum = 45 + 10 = 55

Kết quả:

=== TÍNH TỔNG DÃY SỐ ===
Số hiện tại: 1 - Tổng tạm thời: 1
Số hiện tại: 2 - Tổng tạm thời: 3
Số hiện tại: 3 - Tổng tạm thời: 6
Số hiện tại: 4 - Tổng tạm thời: 10
….
Số hiện tại: 10 - Tổng tạm thời: 55
Tổng cuối cùng từ 1 đến 10 là: 55
