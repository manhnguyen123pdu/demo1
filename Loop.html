GIỚI THIỆU VỀ VÒNG LẶP

2.1 Khái niệm và mục đích của vòng lặp

Định nghĩa: Vòng lặp là cấu trúc điều khiển cho phép thực thi một khối lệnh nhiều lần dựa trên một điều kiện nhất định.

Mục đích của vòng lặp:

  • Tự động hóa: Thực hiện các tác vụ lặp đi lặp lại
  • Xử lý tập hợp: Duyệt qua mảng, danh sách, collection
  • Tính toán lặp: Tính giai thừa, tổng dãy số,...
  • Kiểm tra liên tục: Đợi sự kiện, kiểm tra trạng thái

Các loại vòng lặp trong Java:

  • for - Biết trước số lần lặp
  • while - Không biết trước số lần lặp, kiểm tra điều kiện trước
  • do-while - Không biết trước số lần lặp, thực hiện ít nhất 1 lần
  • for-each - Duyệt qua mảng và collection

Lưu ý quan trọng

Vòng lặp là công cụ mạnh mẽ để xử lý các tác vụ lặp đi lặp lại. Hiểu rõ từng loại vòng lặp và khi nào nên sử dụng chúng sẽ giúp bạn viết mã hiệu quả hơn.

VÒNG LẶP FOR

2.2 Vòng lặp for

Định nghĩa: Vòng lặp for được sử dụng khi biết trước số lần lặp.

Lưu đồ thuật toán:

Cú pháp
for (khởi_tạo; điều_kiện; cập_nhật) {
    // Khối lệnh thực thi
}

Giải thích các thành phần:

  • khởi_tạo: Khai báo và khởi tạo biến đếm (chạy 1 lần duy nhất)
  • điều_kiện: Kiểm tra trước mỗi lần lặp, nếu true thì tiếp tục
  • cập_nhật: Thay đổi biến đếm sau mỗi lần lặp
  • khối lệnh: Các câu lệnh được thực thi trong mỗi vòng lặp
ForLoopBasic.java
public class ForLoopBasic {
    public static void main(String[] args) {
        System.out.println("=== VÒNG LẶP FOR CƠ BẢN ===");
        
        for (int i = 1; i <= 5; i++) {
            System.out.println("Lần lặp thứ: " + i);
        }
        System.out.println("Kết thúc vòng lặp");
    }
}

Giải thích chi tiết luồng thực thi:

  • BƯỚC 1: Khởi tạo - int i = 1 (chạy 1 lần)
  • BƯỚC 2: Kiểm tra điều kiện - i <= 5 → 1 <= 5 → TRUE
  • BƯỚC 3: Thực thi khối lệnh - in "Lần lặp thứ: 1"
  • BƯỚC 4: Cập nhật - i++ → i = 2
  • BƯỚC 5: Kiểm tra điều kiện - i <= 5 → 2 <= 5 → TRUE
  • BƯỚC 6: Thực thi khối lệnh - in "Lần lặp thứ: 2"
  • BƯỚC 7: Cập nhật - i++ → i = 3
  • BƯỚC n: Khi i = 6 → điều kiện 6 <= 5 → FALSE → thoát vòng lặp

Kết quả:
=== VÒNG LẶP FOR CƠ BẢN ===
Lần lặp thứ: 1
Lần lặp thứ: 2
Lần lặp thứ: 3
Lần lặp thứ: 4
Lần lặp thứ: 5
Kết thúc vòng lặp

Ví dụ 2: Tính tổng các số từ 1 đến 10

ForLoopSum.java
public class ForLoopSum {
    public static void main(String[] args) {
        System.out.println("=== TÍNH TỔNG DÃY SỐ ===");
        
        int sum = 0;
        
        for (int i = 1; i <= 10; i++) {
            sum = sum + i; // Hoặc sum += i;
            System.out.println("Số hiện tại: " + i + " - Tổng tạm thời: " + sum);
        }
        
        System.out.println("Tổng cuối cùng từ 1 đến 10 là: " + sum);
    }
}

VÒNG LẶP WHILE

2.3 Vòng lặp while

Định nghĩa: Vòng lặp while được sử dụng khi không biết trước số lần lặp, kiểm tra điều kiện trước khi thực thi.

Lưu đồ thuật toán:

Cú pháp
while (điều_kiện) {
    // Khối lệnh thực thi
    // Cập nhật biến điều kiện
}

Ví dụ 1: Đếm ngược từ 10 đến 1

WhileLoopBasic.java
public class WhileLoopBasic {
    public static void main(String[] args) {
        System.out.println("=== VÒNG LẶP WHILE - ĐẾM NGƯỢC ===");
        int count = 10;
        
        while (count > 0) {
            System.out.println("Đếm: " + count);
            count--;
            // Giảm count đi 1, quan trọng: phải cập nhật biến điều kiện
        }
        System.out.println("Chúc mừng năm mới!");
    }
}

Giải thích chi tiết luồng thực thi:

  • BƯỚC 1: Khởi tạo count = 10
  • BƯỚC 2: Kiểm tra count > 0 → 10 > 0 → TRUE
  • BƯỚC 3: Thực thi - in "Đếm: 10"
  • BƯỚC 4: Cập nhật - count-- → count = 9
  • BƯỚC 5: Kiểm tra count > 0 → 9 > 0 → TRUE
  • BƯỚC 6: Thực thi - in "Đếm: 9"
  • BƯỚC n: Khi count = 0 → điều kiện 0 > 0 → FALSE → thoát vòng lặp

Kết quả:
=== VÒNG LẶP WHILE - ĐẾM NGƯỢC ===
Đếm: 10
Đếm: 9
Đếm: 8
.....
Đếm: 1
Chúc mừng năm mới!

VÒNG LẶP DO-WHILE

2.4 Vòng lặp do-while

Định nghĩa: Vòng lặp do-while thực thi khối lệnh ít nhất 1 lần, sau đó mới kiểm tra điều kiện.

Lưu đồ thuật toán:

Cú pháp
do {
    // Khối lệnh thực thi (ít nhất 1 lần)
    // Cập nhật biến điều kiện
} while (điều_kiện);

Ví dụ: Menu lựa chọn

DoWhileMenu.java
import java.util.Scanner;

public class DoWhileMenu {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int choice;
        
        System.out.println("=== HỆ THỐNG QUẢN LÝ SINH VIÊN ===");
        
        do {
            // Hiển thị menu
            System.out.println("\n===== MENU CHÍNH =====");
            System.out.println("1. Xem danh sách sinh viên");
            System.out.println("2. Thêm sinh viên mới");
            System.out.println("3. Sửa thông tin sinh viên");
            System.out.println("4. Xóa sinh viên");
            System.out.println("5. Thoát chương trình");
            System.out.print("Chọn chức năng (1-5): ");
            
            choice = scanner.nextInt();
            
            // Xử lý lựa chọn
            switch (choice) {
                case 1:
                    System.out.println("Đang hiển thị DS sinh viên...");
                    break;
                case 2:
                    System.out.println("Đang thêm sinh viên mới...");
                    break;
                case 3:
                    System.out.println("Đang sửa thông tin SV...");
                    break;
                case 4:
                    System.out.println("Đang xóa sinh viên...");
                    break;
                case 5:
                    System.out.println("Cảm ơn đã sử dụng! Tạm biệt!");
                    break;
                default:
                    System.out.println("Lựa chọn không hợp lệ!");
            }
            
        } while (choice != 5); // Lặp lại cho đến khi chọn 5 (Thoát)
        scanner.close();
    }
}

Giải thích chi tiết:

  • BƯỚC 1: Hiển thị menu và nhập lựa chọn (luôn chạy ít nhất 1 lần)
  • BƯỚC 2: Xử lý lựa chọn bằng switch-case
  • BƯỚC 3: Kiểm tra điều kiện while (choice != 5)
  • Nếu choice ≠ 5 → tiếp tục vòng lặp
  • Nếu choice = 5 → thoát vòng lặp

2.5 So sánh các loại vòng lặp

Đặc điểm for while do-while
Số lần lặp Biết trước Không biết trước Không biết trước
Kiểm tra điều kiện Trước khi lặp Trước khi lặp Sau khi lặp
Số lần thực thi ít nhất 0 lần 0 lần 1 lần
Khởi tạo biến Trong cú pháp Trước vòng lặp Trước vòng lặp
Cập nhật biến Trong cú pháp Trong thân vòng lặp Trong thân vòng lặp
Ứng dụng Duyệt mảng, tính toán có số lần xác định Game, xử lý sự kiện, đọc file Menu, xác thực đầu vào

Lời khuyên sử dụng:

  • Dùng for khi biết chính xác số lần lặp
  • Dùng while khi không biết số lần lặp và cần kiểm tra điều kiện trước
  • Dùng do-while khi cần thực hiện ít nhất một lần và kiểm tra điều kiện sau

2.6 Bài tập thực hành vòng lặp

Bài tập 1: Tính giai thừa

Factorial.java
public class Factorial {
    public static void main(String[] args) {
        int n = 5;
        long factorial = 1;
        
        for (int i = 1; i <= n; i++) {
            factorial *= i;
            System.out.println(i + "! = " + factorial);
        }
        
        System.out.println("Giai thừa của " + n + " là: " + factorial);
    }
}

Bài tập 2: Kiểm tra số nguyên tố

PrimeNumber.java
public class PrimeNumber {
    public static void main(String[] args) {
        int number = 17;
        boolean isPrime = true;
        
        if (number <= 1) {
            isPrime = false;
        } else {
            for (int i = 2; i <= Math.sqrt(number); i++) {
                if (number % i == 0) {
                    isPrime = false;
                    break;
                }
            }
        }
        
        if (isPrime) {
            System.out.println(number + " là số nguyên tố");
        } else {
            System.out.println(number + " không phải số nguyên tố");
        }
    }
}

2.7 break và continue trong Java

2.7.1 Từ khóa break trong Java

Mục đích: Dùng để thoát khỏi vòng lặp hoặc câu lệnh switch ngay lập tức, trước khi nó kết thúc tự nhiên. Khi gặp break, chương trình sẽ dừng vòng lặp hiện tại và chuyển sang thực hiện lệnh tiếp theo sau vòng lặp.

Lưu đồ thuật toán:

Ví dụ : Thoát vòng lặp khi tìm thấy số chia hết cho 7

BreakExample.java
public class BreakExample {
    public static void main(String[] args) {
        for (int i = 1; i <= 10; i++) {
            if (i % 7 == 0) {
                System.out.println("So chia het cho 7: " + i);
                break; // Thoát khỏi vòng lặp
            }
            System.out.println("Dang kiem tra i = " + i);
        }
        System.out.println("Ket thuc vong lap!");
    }
}

2.7.2 Từ khóa continue trong Java

Mục đích: Dùng để bỏ qua phần còn lại của vòng lặp hiện tại và chuyển sang lần lặp tiếp theo.

Lưu đồ thuật toán:

Ví dụ : Bỏ qua số chẵn trong vòng lặp

ContinueExample.java
public class ContinueExample {
    public static void main(String[] args) {
        for (int i = 1; i <= 10; i++) {
            if (i % 2 == 0) {
                continue; // Bỏ qua các số chẵn
            }
            System.out.println("So le: " + i);
        }
    }
}

Ví dụ tổng hợp break và continue

BreakContinueDemo.java
public class BreakContinueDemo {
    public static void main(String[] args) {
        for (int i = 1; i <= 10; i++) {
            if (i == 3) {
                continue; // Bỏ qua i = 3
            }
            
            if (i == 8) {
                break; // Thoát vòng lặp khi i = 8
            }
            
            System.out.println("i = " + i);
        }
        System.out.println("Ket thuc chuong trinh");
    }
}
